# strava

### Spec:

1. Create Map using GPS
2. Navigate Map
3. Log GPS + Analytics on this Log GPS
4. Minimalist UI
5. Mobile Only

### Features:

1. Battery Saving: Not so accurate, but correct GPS logging
2. Cost Efficient: Mobile  processing  
3. Planning:
   1. Weather and Temperature Analytics
   2. Assisted Transport: Train and Bus for Cycle, Boat
   3. Path counts Database: Popularity, difficulty
   4. Training Plan Manager
   5. Cycle Health Monitoring: Grease
   6. Recce by Car
   7. Safety Rules Manager: Lights, Speed, Food Analytics
   8. Trail-Pieces to Trail-Sequences
   9. Smart Combination of Recipes using Trail-Piece Ingredients
   10. Smart Plan using Path Info
   11. Elevation Statistics
   12. Road Types: Wide roads vs Small Roads
   13. Headwind and Tailwind Analytics


4. Live Analytics:
   1. Traffic Data and Traffic Sensing
   2. Share Live Location Team
   3. Audio Cues
   4. GPS Logging
   5. Navigating
   6. Calorie and Food and Hydration Manager

5. Post-Processing Analytics:
   1. Achievements KOM
   2. Similar People Match
   3. Path Database Repository
   4. Push to insta: templates
   5. Vibration recording and road condition
   6. Add Data from sensors
   7. Classifier: Person's Health, Path Difficulty, Bike Type, Danger Type, Path Beauty Analyzer
   8. Image

6. Social:
   1. Groups via WhatsApp
   2. Emergency Contact and Alarm
   3. Parental Data Sharing
   4. Share and Aggregate Open Repository Paths
   5. Song Download Playlist
   6. Simple API for input and output
